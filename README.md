>已支持Flutter 3.x

## 如何运行

1. 搭建Flutter开发环境，可参考：https://docs.flutter.dev/get-started/install
2. 用Android Studio打开项目
3. 运行`flutter pub get`获取项目依赖
4. 打开Android或iOS模拟器
5. 通过Android Studio 运行按钮运行项目
